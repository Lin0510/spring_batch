## Manager API

### 啟動 Job
#### 啟動 agent1 job
GET http://localhost:8080/start/agent/ID1
#### 啟動 agent2 job
GET http://localhost:8080/start/agent/ID2

### Shutdown Job
#### Shutdown agent1 job
GET http://localhost:8080/stop/agent/ID1
#### Shutdown agent2 job
GET http://localhost:8080/stop/agent/ID2
